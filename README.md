# ❤️ Heart Stroke Prediction Web App

An AI-powered web application that predicts the risk of heart stroke using user health and medical indicators.  
Built with **Python, Streamlit, and Machine Learning**.

---

## 🚀 Features

- 🧠 AI-based heart stroke risk prediction
- 🧍 Personal Details Input (Age, Sex, etc.)
- 🩺 Medical Indicators (Blood Sugar, ECG, etc.)
- 🎨 Clean & modern UI (Streamlit)
- ⚡ Fast and easy to use

---

## 🛠 Tech Stack

- **Python**
- **Streamlit**
- **Pandas**
- **Scikit-learn**
- **Joblib**
